This Section allows user to see all the meter and transformer readings according to timestamp.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The filter for the Dashboard page contain one section. This allows user to filter out the list by timestamp.after selecting the time interval user has click the refresh button. By default for every 2 minutes the list will be refresh.


The Header contains four fields.
1. HS: Shows the Transformer ID.
2. Status: Tells the user whether the device is connected or not.
3. Last Data received: Last data received for displayed Transformer.
4. Connected meters count: Total number of meters are connected to the transformer.
