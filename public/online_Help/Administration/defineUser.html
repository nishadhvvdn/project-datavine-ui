The users list shows every user in the system. The list contains six columns.
1. User ID: column shows the unique user ID of the account.
2. Name: This column shows the name of the person registered to the account.
3. E-Mail Address: This one shows the email associated with the user.
4. Security Group: This one shows the security group that is hooked up to this account.
5. Account Locked: This column shows whether or not the account has been locked.
6. Action: This column contains buttons for editing or deleting the account and one for reseting the user’s password

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Security Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The filter for the users page contains only one section. This consists of a single checkbox to filter out locked accounts.

To search for a user simply click the tab corresponding to user would like to look up. Then enter the entire User Id or a portion of it and click search button for pull up matching user.

To add a user click the Add User button above the top right corner of the list.

Clicking the button will open the user creation window. From here simply enter the user details for the account user wish to create and click OK. The new account will be added immediately to the list.

To edit a user locate the account in the list and click the edit button in the Action column.

The edit window will open after clicking the edit button. From here change any values user wish and then click the OK button to save user changes.

To delete a user locate the user on the list and click the delete button in the Action column. When prompted if user are sure click yes and the user will be deleted.

Confirmation Popup For Delete User.

To reset a user’s password locate the user on the list and click the reset password button in the Action column. When prompted if user are sure click yes and the user’s password will be reset and emailed to the email associated with account.

To view a User’s details click particular User Id from the list. This will pull up a window with the User details.
