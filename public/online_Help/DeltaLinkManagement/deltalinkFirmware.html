The firmware management list shows all current and historical firmware downloads. The list contains seven columns.
1.Job ID: This column shows unique ID of the job.
2.Status:This column shows whether the firmware update is pending, running, completed, or failed.
3.Card Type: This column shows Card type of the job.
4.Version: The version being of the firmware being installed.
5.Group Name: The group that the firmware is being downloaded to.
6.Download Date Timestamp: The time the CollectionENGINE™ started the job.
7.Action: The action column contains actions user can take on the job such as beginning it.

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The filter for the firmware management page allows user to remove all but running jobs from the list by checking the related checkbox.

A user is given different actions to perform on a firmware download based on where the job is in its download process. For instance, user would see a resend option if the job failed. To start a job action simply scroll to the appropriate entry in the list and click whichever action user wish to take in the Firmware Job Actions column.

To begin importing a new firmware file into the system, simply click the Import Firmware button above the upper right corner of the firmware jobs list.

After clicking the button the import window will open. Simply choose the file user wish to import by clicking the Choose File button, check make production firmware if user wish to mark it as production firmware, and click Import. The result will be shown on the Import Result line.

The begin downloading new firmware to devices user must begin the download mode on the devices. To do this click the Enter Firmware Download Mode above the right corner of the list.

A window will appear after clicking the button. Select a firmware file on the system (Note, the plus button next to the firmware dropdown will take user to the firmware import screen). Then select the group to download and click Enter Firmware Download Mode. The job will appear in the list and the user can begin the download from the list entry.
