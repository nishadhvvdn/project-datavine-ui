The DeltaLINK™ Management page is the hub for all things DeltaLINK™ related. This portion of the system allows users to manage configuration files and firmware files currently on the delta links. Furthermore a user can control security settings of the delta links and the groups they belong to as well as view all currently running DeltaLINK™ jobs.

Group
Management
This section allows a user to control what groups the delta links belong to on the CollectionENGINE™.

Management
This section allows a user to view and update firmware on the delta links.
