The config program list shows every HyperSPROUT™ configuration program that has been added to the system.
The list contains four columns.
1. Name: This column contains the unique name of the configuration program.
2. Description: This column shows a brief description of what the configuration program is for.
3. Members: This column shows how many configuration groups are using the particular configuration program .
4. Action: This column contains a button for creating new configuration program from the selected ones defaults and a deletion button for removing the program.

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Name header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.

Uploading a config program consists of three steps.
The first is clicking the Upload Config Program button located above the upper right corner of the list.

After that the user will be presented with a small window. This window will ask for the unique name the user wishes to call the program, a description, and the path to the program file.The path is inputed by clicking the choose file button.

Finally, the user must click the save button after which the new program will be added to the list.

To update a config program simply locate the config program in the list and click the Save As button in the Action column.

After clicking the button the user will be shown the Save As window. It will be immediately populated with the values of the configuration program from the row the user selected. From here any value can be changed. Once you are happy with the changes input a name and description and click the save button. This will create a new configuration program with the selected values.

To delete a configuration program simply locate the configuration program you wish to delete in the list, click the delete button in the action column, and select yes when prompted if you are sure. Note, this button is only available if no configuration groups are using the config program.
