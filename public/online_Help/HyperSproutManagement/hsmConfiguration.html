The configurations list shows every HyperSPROUT™ configuration group that has been added to the system.
The list contains seven columns.
1. Name: This column contains the unique name of the configuration group.
2. Description: This column shows a brief description of what the configuration group is for.
3. Members: This column shows how many HyperSprouts are using the particular configuration group.
4. Device Class: The device class of the HyperSprouts belonging to the group.
5. Config Program: The name of the config program associated with the configuration group
6. Edit Time: The timestamp for when the group was last edited
7. Action: This column contains a button for editing the configuration group and a deletion button for removing the group.

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Name header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The list gives four different options for exporting. The user may export either all the data in the list or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the list after filtering it for the desired data.

The list can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.

The filter for the configurations page only contains one section. This section allows a user to filter the report so that only entries containing an edit time within the user specified date range will be shown.

Creating a configuration group consists of three steps. The first is clicking the New button located above the upper right corner of the list.

After that the user will be presented with a small window. This window will ask for the unique name the user wishes to call the group, the device class, the configuration to associate with the group, and a description.

Finally, the user must click the save button after which the new group will be added to the list.

To view a groups basic details a user simply has to locate it in the list and click the name after which a window will appear showing the group’s details.

To change the configuration associated with a group first find the group within the list and then click the edit button in the action column.

The edit configuration window will pop up. From here you can select any of the config programs in the system from the config program dropdown. Then just click Save to change the associated configuration program (Note, this only changes the groups configuration program on the CollectionENGINE™. To have it actually change on the associated HyperSprouts you must download the configuration program to the devices from the Downloads tab).

To delete a configuration simply find the group you wish to delete in the list, click the delete button in the in the action column, and select yes when prompted if you are sure.

To add members to a group you must first click the Add members button above the top right corner of the configuration groups list.

This will open up a creation window. Input a unique name and a brief description of the application group’s purpose then click save. A new application group matching the inputed information will immediately be added to the list.
