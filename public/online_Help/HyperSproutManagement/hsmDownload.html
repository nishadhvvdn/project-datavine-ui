The downloads list shows every HyperSPROUT™ download job in the system.
The list contains seven columns.
1. Job ID: This column contains the unique id of the job.
2. Description: This column shows a brief description of what the job is for.
3. Group: Column shows what group activated the download.
4. Serial Number: Column shows serial number of the device.
5. Status: Status shows if the download is in progress or completed among all members in the group.
6. Start Time: Timestamp for when the download started.
7. End Time: Timestamp for when the download ended.

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.

The filter for the downloads page only contains one section. This section allows a user to filter the report so that only entries containing a start time within the user specified date range will be shown.
