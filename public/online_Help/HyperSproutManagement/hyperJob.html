The HyperSPROUT™ job status list shows every job that has or is running on the HyperSprouts in the system.
The list contains seven columns.
1.Job ID: This column shows unique ID of the job.
2.Description: This column shows a brief description of what the job is.
3.Status: This column shows whether or not the job is currently running.
4.Job Name: This column shows name of the job.
5.Serial Number: This column shows Serial number of the job.
6.Start Time: The timestamp for when the job began.
7.End Time: The timestamp for when the job Ends.

The report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the HyperSPROUT™ job status page contains two sections.
The first allows user to check what types of jobs user want to display and whether or not user want to display finished jobs.

The second section allows user to input a date range that will filter the list for jobs that were started within the specified range.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.
