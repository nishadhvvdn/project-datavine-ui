As an admin if he wants do operation for multiple meters for read/connect/disconnect then this functionality will come into the picture. Then Bulk Operation functionality under meter management will help the admin. With this functionality when admin send request to multiple meters via uploading a file once cloud receive the response it will generate the report.
The list contains two columns.
1.Name : This column shows Name of the operation.
2.Date and Time: The timestamp for when the operation began.

The report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The section allows user to input a date range that will filter the list for jobs that were started within the specified range.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

To begin importing a new meters file into the system, simply click the Bulk Opeartion button above the upper right corner of the bulk opeartion list.

After clicking the button the import window will open. Simply choose the file user wish to import by clicking the Choose File button, select operation name which user wants to perform , and click Import. The result will be shown on the Import Result line.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

User can download the bulk operation file from Action column in Grid list.
