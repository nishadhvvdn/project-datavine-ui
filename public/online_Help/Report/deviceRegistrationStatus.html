The Device Registration Status report shows all devices that have been deregistered from the CollectionENGINE™. The report contains four columns.
1.Serial Number: The serial number of the device that was deregistered.
2.Type:The type of device is shown here. This can be a HyperSPROUT™ or meter.
3.Last Deregistered Time: The timestamp of when the device was deregistered should be here.
4.HyperSPROUT™: The serial number of the HyperSPROUT™ that was collecting data from the device before it deregistered.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The filter for the device registration status report contains one section. It allows the user to input a threshold for the deregistration time column by entering a date and time.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Last Deregistered Time header will switch the column between the default display, ascending times, and descending times.
