The new account report shows all accounts created within the user specified time period(30 days).The report contains five columns.
1. Serial Number : This column contains serial number of the newly added meter device.
2. Address:The Address column shows the location where the new meter have been added.
3. Account Name: This column shows the name on the account.
4. Meter ID:The report will show the ID associated with the meter.
5. Installation Date: This column shows the time and date at which the account has been added.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the new account report contains time filter. This section allows a user to select the date span of of new record to show.The user is provided with a dropdown containing common date spans to select from such as the previous week.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the address header will switch the column between the default display, ascending event times, and descending event times.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.
