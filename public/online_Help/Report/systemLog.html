The system log report shows all events the system received over a user specified time period. The report contains four columns.
1. HS Serial Number: This column shows the HyperSPROUT™ serial number of the device associated with the event.
2. Meter Serial Number: This column shows the meter serial number of the device associated with the event.
3. Event Date/Time: This column contains the date and time at which the event was recorded.
4. Message: The message associated with the event will be shown here.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The first section allows a user to search the specific devices they would like to view by their serial number rather than the default of all devices.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Event Date/Time header will switch the column between the default display, ascending event times, and descending event times.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.
