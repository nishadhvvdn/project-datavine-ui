2.   Changing The Password

To change the account settings simply change the value of the setting you want to alter in the text box associated with the setting and then click save

To change the password of the account begin by clicking the Change Password button in the bottom right.

This will open a new window prompting for the current account password. Input this along with the new password and click OK. This will change the password for the account.
