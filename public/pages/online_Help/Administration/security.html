The security list shows every every security group in the system. The list contains four columns.
1. Security Group ID:The unique ID of the security group will be shown here.
2. Description:A brief description of the security group will be displayed in this column.
3. No of Users: This column shows the number of users attached to the security group.
4. Action: This column contains buttons for editing and deleting a security group.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Description header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

To change global password settings such as the number of days a password can remain active before needing to change click the Password Settings button above the top right corner of the list.

To add a new security group simply click the Add Security Group button above the top right corner of the list.

The creation window will pop up after the button is clicked. From here simply enter the security group ID, the description, and checkmark the areas users attached to this group should be able to access. Click OK and the security group will be added.

To edit a security group locate it on the list and click the edit button in the Action column of the security group you wish to change.

A window will appear where you can change any of the values in the security group. After you have finished click OK to save the changes to the group.

To delete a security group locate it on the list and click the delete button in the Action column. When prompted if you are sure click yes and the group will be deleted.
