The firmware management list shows all current and historical firmware downloads. The list contains twelve columns.
1.Job ID: This column shows unique ID of the job.
2.Status:This column shows whether the firmware update is pending, running, completed, or failed.
3.Phase: : This one shows if the download job is in the initial, resend, or recovery phase of the download.
4.Description: A brief description of the job.
5.Type: The type of device that the firmware is being installed on.
6.Version: The version being of the firmware being installed.
7.Group: The group that the firmware is being downloaded to.
8.Submit Time: The time the CollectionENGINE™ started the job.
9.% Complete: The percentage of the job that has been completed (Note, this includes failed downloads as long as the attempt has completed).
10.% Succesful: The percentage of the devices that have successfully downloaded the new firmware.
11.Firmware Job Actions: : Depending on the status of the job this column will have buttons to take appropriate action. For instance, if the job has failed a resend firmware action will be available.
12.Action: The action column contains actions you can take on the job such as beginning it.

If the list has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The filter for the firmware management page allows you to remove all but running jobs from the list by checking the related checkbox.

A user is given different actions to perform on a firmware download based on where the job is in its download process. For instance, you would see a resend option if the job failed. To start a job action simply scroll to the appropriate entry in the list and click whichever action you wish to take in the Firmware Job Actions column.

To begin importing a new firmware file into the system, simply click the Import Firmware button above the upper right corner of the firmware jobs list.

After clicking the button the import window will open. Simply choose the file you wish to import by clicking the Choose File button, check make production firmware if you wish to mark it as production firmware, and click Import. The result will be shown on the Import Result line.

The begin downloading new firmware to devices you must begin the download mode on the devices. To do this click the Enter Firmware Download Mode above the right corner of the list.

A window will appear after clicking the button. Select a firmware file on the system (Note, the plus button next to the firmware dropdown will take you to the firmware import screen). Then select the group to download and click Enter Firmware Download Mode. The job will appear in the list and the user can begin the download from the list entry.
