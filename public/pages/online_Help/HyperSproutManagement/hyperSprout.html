The HyperSPROUT™ Management page is the hub for all things HyperSPROUT™ related. This portion of the system allows users to manage configuration files and firmware files currently on the HyperSprouts. Furthermore a user can control security settings of the HyperSprouts and the groups they belong to as well as view all currently running HyperSPROUT™ jobs.

Group
Management
This section allows a user to control what groups the HyperSprouts belong to on the CollectionENGINE™.
Firmware
Management
This section allows a user to view and update firmware on the HyperSprouts.
HyperSPROUT™
Job Status
This section gives an overview of all HyperSPROUT™ jobs on the system.
