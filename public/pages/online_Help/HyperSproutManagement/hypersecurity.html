2.   Set Encrpytion Keys

To assign new security codes to a device class you must first click the Assign Device Security Codes button.

Clicking this button will open up the assigning window. From here select the device class you wish to update and the security level from the dropdowns. Then click the Choose File button to input the path to the security codes file and, finally, click Import. The result will be shown on the Import Result line.

To set the encryption keys for a device class first choose the device class from the dropdown in the Select Device Class section.

Then change the values as needed in the Encryption Key Settings section and click Save.
