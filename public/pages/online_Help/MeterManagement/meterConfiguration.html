The Configuration Management page controls all aspects of the systems meter configurations. It allows the creation of new programs and the casting of these to any registered meters. From the contained sections a user can view all current configuration downloads and discrepancies between actual meter settings and expected settings. A user can also create configuration groups and add meters with the same configurations to them from here.

Config Program
This section allows a user to add new configuration programs to the system.
Configurations
This section allows a user to create configuration groups and add meters to it.
Downloads
This section allows a user to download new configurations to groups and view the currently running download jobs.
Tag Discrepancies
This section reports all configuration values that were not updated and matched correctly.
