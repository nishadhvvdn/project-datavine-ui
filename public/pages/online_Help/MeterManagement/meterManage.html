The Meter Management page is the hub for all things meter related. This portion of the system allows users to manage configuration files and firmware files currently on the meters. Furthermore a user can control security settings of the meters and the groups they belong to as well as view all currently running meter jobs.

Group
Management
This section allows a user to control what groups the meters belong to on the CollectionENGINE™.
Firmware
Management
This section allows a user to view and update firmware on the meters.
Meter
Job Status
This section gives an overview of all meter jobs on the system.
