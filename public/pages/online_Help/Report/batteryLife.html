The battery life report shows how long devices have run on a single battery charge. The report contains three columns.
1. Serial Number: This column shows the serial number of the device associated with the battery.
2. Type:The type of device the battery is contained in will be shown here.
3. Minutes On Battery: This column shows how many minutes the device has been running since the battery was last replaces or charged.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the communication statistics report contains two sections. The first allows a user to select the specific devices they would like to view rather than the default of all devices. To open the device selection window a user just has to click the Add button.

The user will then be able to select the device or devices they would like to be displayed.

The user can then further filter the report with the second section. From this section a user can choose to display records where the last read time is prior to whatever datetime the user inputs.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Last Read Time header will switch the column between the default display, ascending read times, and descending read times.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.
