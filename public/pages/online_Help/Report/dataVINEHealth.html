The DataVINE™ health report shows the network health statistics of each device in the system. The report contains six columns.
1. Serial Number: This column contains the device’s serial number.
2. Health Status:This column shows the current Health Status based on any recorded fluctuations in throughput or contact success rate the current month compared to the last twelve (This can be OK, Warning, Error, or Unknown).
3. Current Month’s Avg. Throughput: This will show the average throughput over this month for the device
4. Last Year’s Avg. Throughput: This will show the average throughput over the last twelve calendar months for the device.
5. Current Month’s Contact Success Rate:This will show the percentage of successful contact attempts made with the device for the current month.
6. Last Year’s Avg. Contact Success Rate: This will show the percentage of successful contact attempts made with the device.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the DataVINE™ health report contains two sections. The first allows a user to select the specific devices they would like to view rather than the default of all devices. To open the device selection window a user just has to click the Select Endpoints button.

The user will then be able to select the device or devices they would like to be displayed.

The user can then further filter the report with the second section. From this section a user can choose to display devices based on the current Health Status by checkmarking the types of statuses to display.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Serial Number header will switch the column between the default display, ascending serial numbers, and descending serial numbers.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.
