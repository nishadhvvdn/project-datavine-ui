The reports section provides users with a myriad of different reporting options. A user can see any modifications other users have made, the battery life for all devices in the system, the communication status of interconnected devices and so much more. The reports all include the ability to export the report, print it and filter for specifics. Below each report is described in brief. Please use either the navigation bar or directly click on the heading under circle icon for the relevant section to be taken to the appropriate page.

Communication
Statistics
Use Communication Statistics to monitor device communication events (e.g., reads, interrogations) up to a particular date and time.
System Log
Use System Log to view all system events within a specified time period.
Device
Firmware Versions
Use Firmware Versions to view all firmware update files known to the system and the versions associated with the endpoints.
 
New Accounts
Use New Accounts to view all last 30 days newly updated record of meters.
System Audit Log
Use System Audit Log to view details regarding the events and users.
