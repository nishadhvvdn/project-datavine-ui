The system audit log report shows all user actions on the CollectionENGINE™. The report contains three columns.
1.User ID: The id of the user who carried out the action.
2.Event Date/Time: The timestamp for when the action was done.
3.Event: A brief description of what action was taken on the CollectionENGINE™.

The report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the system audit log report contains three sections.
The first section is a set of checkboxes that allows the user to select which users to filter for.

The second section allows a user to select a date time range for the Event Date/Time column. The user may either use the dropdown to select a common date span such as the last week or the user can use the date and time selection boxes to set a custom date time span.

To sort the report, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Event Date/Time header will switch the column between the default display, ascending times, and descending times.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.
