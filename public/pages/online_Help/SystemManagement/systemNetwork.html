The Network Statistics shows every device that have communicated with the system.
The list contains four columns.
1.Serial Number: This column shows unique serial number of the device.
2.Name:This column shows the name given to the device.
3.Hardware Version: :: This column shows the version number of the hardware on the device.
4.Registered: This column shows whether or not the device has fully registered with the system.

If the report has a large amount of data, it might separate into pages. In this case, the user can choose how many entries to display or which page to show from the pagination tool bar in the bottom left of the report.

The filter for the device management page contains only one section. This is a checkbox to remove or show non-registered devices from the list.

The second section is a time filter. This section allows a user to filter the list so that only entries containing a start time within the user specified date range will be shown.

To sort the list, the user must click the header of whichever column they would like to sort by. For instance, clicking on the Group header will switch the column between the default display, ascending alphabetically, and descending alphabetically.

The report gives four different options for exporting. The user may export either all the data in the report or just the currently displayed data in either a CSV format or a PDF format. To do this the user simply has to click on the export button in the top right corner of the report after filtering it for the desired data.

The report can easily be prepared for printing by using the provided printing button. This button will open up a standard printing window with the report displayed.
