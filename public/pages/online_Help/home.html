Reports

The Reports Section allows a user to create a number of on-demand reports ranging from communication statistics to an audit of every user action on the system.
HyperSPROUT™
Management
The HyperSPROUT™ Management section allows a user to assign new configuration programs, security codes, firmware and manage grouping of similar HyperSprouts.
Meter
Management
The Meter Management section allows a user to assign new configuration programs, security codes, firmware and manage grouping of similar meters.
System
Management
The System Management Section allows a user to register/deregister devices, view the current status of running jobs, clean up endpoints and monitor the DataVINE™ Health.
Tools

The Tools section allows a user to change their account settings.
Administration

The Administration section allows a user to create security groups, users and modify system settings.
