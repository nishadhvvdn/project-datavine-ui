Setting Description Value
Days between TOU pending season retries

Determines how many days a TOU pending season job needs to run before doing point-to-point retries to the unfinished endpoints. 1 to 100.

Node ping period duration in seconds

This setting together with the MaxNodePingsPerPeriod setting define the maximum allowed pace of node ping requests to a single relay for the given window period. Requests that exceed this limit will be rejected. 1 to 86400.
Process TOU pending season updates

Determines whether or not TOU pending season updates will be run automatically when the automated process runs.
The number of simultaneous configuration download jobs allowed

The maximum number of simultaneously active configuration download jobs allowed. 1 to 2.147484E+09.
